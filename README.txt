How to run the application:

1. Download Unity hub
2. Download Unity Editor 2021.3.4f1
3. Open unity editor and click on open project button
4. After clicking on it, select project from disk.
5. Select BTP-B22HS01 from the downloaded location.
6. Select the appropriate unity editor version.
7. Open the project in the unity editor, The application is now started.
8. Visit unity website for more setup tutorials.
